   For each major release of CockroachDB, Cockroach Labs provides maintenance
   support for at least 365 days and assistance support for at least an
   additional 180 days. For more details, see the Release Support Policy.

   If you're having an issue with CockroachDB, you can reach out for support
   from Cockroach Labs and our community:

     * Troubleshooting documentation
     * CockroachDB Community Forum
     * CockroachDB Community Slack
     * StackOverflow
     * File a GitHub issue
     * CockroachDB Support Portal

   We also rely on contributions from users like you. If you know how to help
   users who might be struggling with a problem, we hope you will!
